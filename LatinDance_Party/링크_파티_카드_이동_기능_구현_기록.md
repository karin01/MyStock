# 링크 파티 카드 이동 기능 구현 기록

## 🎯 요구사항
사용자가 링크를 타고 들어오면 해당 파티의 **카드로 바로 이동**하고 **카드를 강조 표시**하는 기능을 구현했습니다.

### 기존 동작
- 링크 접속 시 → 파티 모달만 열림
- 사용자가 어떤 파티인지 파악하기 어려움

### 개선된 동작
- 링크 접속 시 → 파티 카드로 스크롤 이동 → 카드 강조 표시 → 모달 열기
- 사용자가 해당 파티를 쉽게 찾을 수 있음

## 🔧 구현 내용

### 3. 카드 스크롤 이동 및 강조 함수
```javascript
// 파티 카드로 스크롤 이동 및 강조 표시
function scrollToPartyCard(partyId) {
    console.log('파티 카드로 스크롤 이동 시도:', partyId);
    
    // 모든 파티 카드에서 강조 표시 제거
    const allCards = document.querySelectorAll('.party-card');
    allCards.forEach(card => {
        card.classList.remove('highlighted');
    });
    
    // 해당 파티 카드 찾기
    const partyCard = document.querySelector(`[data-party-id="${partyId}"]`);
    
    if (partyCard) {
        console.log('파티 카드 찾음, 스크롤 이동 및 강조 표시');
        
        // 파티 섹션으로 스크롤 이동
        const partiesSection = document.getElementById('parties-section');
        if (partiesSection) {
            partiesSection.scrollIntoView({ behavior: 'smooth', block: 'start' });
        }
        
        // 잠시 후 카드로 스크롤 이동
        setTimeout(() => {
            partyCard.scrollIntoView({ behavior: 'smooth', block: 'center' });
            
            // 강조 표시 추가
            partyCard.classList.add('highlighted');
            
            // 3초 후 강조 표시 제거
            setTimeout(() => {
                partyCard.classList.remove('highlighted');
            }, 3000);
            
        }, 500);
        
    } else {
        console.warn('파티 카드를 찾을 수 없음:', partyId);
        // 카드를 찾을 수 없으면 파티 섹션으로만 이동
        const partiesSection = document.getElementById('parties-section');
        if (partiesSection) {
            partiesSection.scrollIntoView({ behavior: 'smooth', block: 'start' });
        }
    }
}
```

### 4. URL 파라미터 처리 함수 수정

```javascript
async function openPartyModalFromUrl(partyId) {
    console.log('URL 파라미터로 파티 모달 열기 시도:', partyId);
    
    // 파티 정보 찾기
    const party = getPartyById(partyId);
    
    if (party) {
        console.log('파티 정보 찾음, 카드로 이동 및 모달 열기:', party);
        
        // 메타 태그 업데이트 (카카오톡 공유용)
        updateMetaTags(party);
        
        // 파티 카드로 스크롤 이동 및 강조 표시
        scrollToPartyCard(partyId);
        
        // 잠시 후 모달 열기 (카드 강조 효과를 볼 수 있도록)
        setTimeout(async () => {
            await viewParty(partyId);
        }, 1500);
        
        // URL에서 파라미터 제거 (선택사항)
        const newUrl = window.location.pathname;
        window.history.replaceState({}, document.title, newUrl);
        
    } else {
        console.error('URL 파라미터의 파티를 찾을 수 없음:', partyId);
        showMessage('요청하신 파티를 찾을 수 없습니다.', 'error');
    }
}
```

## 🎨 사용자 경험 개선

### 시각적 피드백
- **부드러운 스크롤**: `scroll-behavior: smooth`로 자연스러운 이동
- **카드 강조**: 빨간색 테두리와 그림자로 해당 카드 강조
- **애니메이션 효과**: 2초간 펄스 애니메이션으로 주목도 향상
- **자동 제거**: 3초 후 강조 효과 자동 제거

### 타이밍 최적화
1. **즉시 스크롤**: 파티 섹션으로 먼저 이동
2. **0.5초 후**: 해당 카드로 정확히 이동
3. **1.5초 후**: 모달 열기 (강조 효과를 볼 수 있도록)

### 오류 처리
- **카드 없음**: 파티 섹션으로만 이동
- **파티 없음**: 명확한 오류 메시지 표시
- **강조 제거**: 모든 카드에서 기존 강조 효과 제거

```

3. 해당 파티 카드로 정확히 이동
4. 카드 강조 표시 (빨간 테두리 + 그림자)

```

```

0초: 1.05배 확대 + 밝은 그림자

```
