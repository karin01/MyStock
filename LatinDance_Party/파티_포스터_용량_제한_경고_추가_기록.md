# 파티 포스터 용량 제한 경고 추가 기록

## 📅 작업 일자
2024년 12월 19일

## 🎯 작업 목표
파티 포스터 업로드 시 사용자에게 1MB 이하 파일 크기 제한을 알리는 경고 메시지 추가

## 🔧 구현 내용

### 1. **HTML 구조 추가**
```html
<div class="form-group">
    <label for="party-poster">파티 포스터:</label>
    <input type="file" id="party-poster" name="file" accept="image/*">
    <div class="file-size-warning">
        <span class="warning-icon">⚠️</span>
        <span class="warning-text">파일 크기는 1MB 이하로 업로드해주세요. (큰 파일은 자동으로 압축됩니다)</span>
    </div>
    <div id="poster-preview"></div>
</div>
```

### 2. **CSS 스타일 추가**
```css
/* 파일 크기 경고 메시지 */
.file-size-warning {
    display: flex;
    align-items: center;
    gap: 8px;
    margin-top: 8px;
    padding: 8px 12px;
    background: linear-gradient(135deg, #FFF3CD, #FFEAA7);
    border: 1px solid #FFC107;
    border-radius: 8px;
    font-size: 13px;
    color: #856404;
    box-shadow: 0 2px 4px rgba(255, 193, 7, 0.1);
}

.warning-icon {
    font-size: 16px;
    flex-shrink: 0;
}

.warning-text {
    line-height: 1.4;
    word-break: keep-all;
}
```

### 3. **모바일 반응형 스타일**
```css
@media (max-width: 768px) {
    .file-size-warning {
        padding: 10px 14px;
        font-size: 14px;
        margin-top: 10px;
    }
    
    .warning-icon {
        font-size: 18px;
    }
    
    .warning-text {
        font-size: 13px;
    }
}
```

## 🎨 디자인 특징

### **시각적 요소**
- **경고 아이콘**: ⚠️ 이모지 사용
- **배경색**: 노란색 그라데이션 (#FFF3CD → #FFEAA7)
- **테두리**: 주황색 (#FFC107)
- **그림자**: 부드러운 그림자 효과

### **사용자 경험**
- **명확한 메시지**: "파일 크기는 1MB 이하로 업로드해주세요"
- **안심 메시지**: "(큰 파일은 자동으로 압축됩니다)"
- **모바일 최적화**: 작은 화면에서도 가독성 확보
